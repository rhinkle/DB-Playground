# Database Playground

### Description
This is a playground where you can test out your SQL skills. Any edits made to your database will persist to the `opt` dir.

### Quick Start

#### You will need
* Docker
* Internet
* Cats

#### Commands

Starts your database
```
docker-compose up -d
```

Stop your database
```
docker-compose stop
```

#### Phpmyadmin
phpmyadmin is a GUI for mysql.
* URL: localhost:8080
* Username: root
* Password: helloworld
